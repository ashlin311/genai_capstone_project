Getting Started Guide

  1. Setup LLM Keys: Ensure that your Groq API key is set up in a .env file in the project folder root.
  2. Build Vectorstore: If the Vector Store status shows "Unloaded", run python ingest.py in your project directory to build the database from local text archives.
  3. Ask Questions: Navigate to the RAG Chatbot to query general topics like Aptitude, HR Questions, or TCS Placement logs.
  4. Upload Job Descriptions: Head over to the JD Prep Planner, drag in a PDF JD, and watch the pipeline build a custom preparation study guide.
